
CREAR UN NUEVO REPOSITORIO O TRAER UNO EXISTENTE

…or create a new repository on the command line

    echo "# repositorio" >> README.md
    git init
    git add README.md
    git commit -m "first commit"
    git branch -M main
    git remote add origin https://github.com/DanielLorenzoCabrera/repositorio.git
    git push -u origin main

    …or push an existing repository from the command line

    git remote add origin https://github.com/DanielLorenzoCabrera/repositorio.git
    git branch -M main
    git push -u origin main
